# Enquiry Form

## Overview
Textarea and submit button for pasting and analysing client enquiries.

## Props
| Prop | Type | Description |
|------|------|-------------|
| `onSubmit` | (text: string) => void | Called with text when form is submitted |
| `loading` | boolean | Disables input and button while processing |
