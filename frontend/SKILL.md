# Frontend

## Overview
React application (Vite + Tailwind) that provides the user interface for submitting enquiries, viewing classification results, and browsing history.

## Skills

| Skill | Location | Purpose |
|-------|----------|---------|
| enquiry-form | `src/skills/enquiry-form/` | Text input and submit button |
| result-card | `src/skills/result-card/` | Displays classification result, confidence, entities, suggested response |
| enquiry-history | `src/skills/enquiry-history/` | Sidebar listing past analyses |
| shared | `src/skills/shared/` | Shared UI components (ErrorBanner, LoadingSpinner) |
